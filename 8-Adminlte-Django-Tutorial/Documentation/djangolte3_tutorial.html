1 — Create Project Environment

Create and activate a virtual environment:

virtualenv venv
.\venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux
2 — Install Django & django-adminlte3
pip install django django-adminlte3
pip install django-adminlte3-admin  # Optional for Admin styling
3 — Create a New Django Project and App
django-admin startproject DemoAdminLTE3 .
py manage.py startapp DemoApp
py manage.py makemigrations
py manage.py migrate
4 — Create a Superuser
py manage.py createsuperuser
py manage.py runserver

Visit:

  • App root → http://127.0.0.1:8000/
  • Admin → http://127.0.0.1:8000/admin/
5 — Enable django-adminlte3 Templates

Add to INSTALLED_APPS in settings.py:

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    
    'DemoApp',
    
    'adminlte3',
    'adminlte3_theme',
]

Ensure your template directory is enabled:

'DIRS': [BASE_DIR / "templates"],   # Important
6 — Create a Simple View Using AdminLTE Base Template

Create DemoApp/templates/demoapp/home.html:

{% raw %}{% extends "adminlte/base.html" %}

{% block content %}


    
Welcome to AdminLTE Demo

    
This page uses the AdminLTE3 layout!



{% endblock %}{% endraw %}

View in DemoApp/views.py:

from django.shortcuts import render

def home(request):
    return render(request, 'demoapp/home.html')
7 — Add URL Routing

Create DemoApp/urls.py:

from django.urls import path
from .views import home

urlpatterns = [
    path('', home, name='home'),
]

Include in DemoAdminLTE3/urls.py:

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('DemoApp.urls')),
]
8 — How Template Inheritance Works

Template hierarchy:

home.html
  ↓ extends
adminlte/base.html
  ↓ includes
_main_header.html
_main_sidebar.html
_main_footer.html

Django allows overriding any AdminLTE template by recreating its folder structure in templates/.

9 — Override the Sidebar

Original template location:

adminlte3/templates/adminlte/lib/_main_sidebar.html

Override in project:

project_root/
    templates/
        adminlte/
            lib/
                _main_sidebar.html

Supports nested blocks: logo, user_panel, nav_links.

10 — Override Header and Footer

    Y — Django REST Framework Integration

    Install DRF:

    pip install djangorestframework

    Add to INSTALLED_APPS and create serializers/views:

    from rest_framework import serializers, viewsets
from .models import Item

class ItemSerializer(serializers.ModelSerializer):
    class Meta:
        model = Item
        fields = '__all__'

class ItemViewSet(viewsets.ModelViewSet):
    queryset = Item.objects.all()
    serializer_class = ItemSerializer
    Z — Deployment with Nginx + Gunicorn

    Example Gunicorn start:

    gunicorn DemoAdminLTE3.wsgi:application --bind 0.0.0.0:8000

    Sample Nginx server block:

    server {
    listen 80;
    server_name yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
