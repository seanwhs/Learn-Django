# ğŸ“Œ **2. Rendering a Template in a View**

`app/views.py`

```python

    return render(request, 'app/home.html')
```

`mysite/urls.py`

```python

]
```

# ğŸ“Œ **3. Template Variables**

Pass data from the view to the template:

### View:

```python
def home(request):
    context = {'user_name': 'Alice'}
    return render(request, 'app/home.html', context)
```

### Template:

```html
<h1>Welcome, {{ user_name }}!</h1>
```

Variables are surrounded by **double curly braces**: `{{ ... }}`.

# ğŸ“Œ **4. Template Tags**

Tags let you add logic to templates.

### Example: `if`

```html
{% if user_name %}
    <p>Hello, {{ user_name }}!</p>
{% else %}
    <p>Guest User</p>
{% endif %}
```

### Example: `for`

```html
<ul>
{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}
</ul>
```

---

# ğŸ“Œ **5. Template Filters**

Filters transform values.

Examples:

```html
{{ name|upper }}
{{ description|truncatewords:10 }}
{{ price|floatformat:2 }}
{{ date|date:"F d, Y" }}
```

Common filters:

| Filter    | Example  | Output         |                 |
| --------- | -------- | -------------- | --------------- |
| `upper`   | `"hello" | upper`         | `"HELLO"`       |
| `lower`   | `"Hi"    | lower`         | `"hi"`          |
| `length`  | `list    | length`        | number of items |
| `default` | `value   | default:"N/A"` | N/A if empty    |

# ğŸ“Œ **6. Template Inheritance (VERY IMPORTANT)**

This avoids repeating HTML on every page.

---

### Step 1: Create a base template

`templates/base.html`

```html
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}My Site{% endblock %}</title>
</head>
<body>

<nav>
    <a href="/">Home</a>
    <a href="/about/">About</a>
</nav>

<hr>

{% block content %}
{% endblock %}

</body>
</html>
```

---

### Step 2: Extend the base template

`templates/app/home.html`

```html
{% extends "base.html" %}

{% block title %}Homepage{% endblock %}

{% block content %}
<h1>Welcome Home</h1>
<p>This is the home page.</p>
{% endblock %}
```

# ğŸ“Œ **8. Static Files in Templates (CSS, JS, Images)**

Enable static files:

In `settings.py`:

```python
STATIC_URL = "static/"
STATICFILES_DIRS = [BASE_DIR / "static"]
```

Folder:

```
static/
    css/style.css
```

Load and use in templates:

```html
{% load static %}

<link rel="stylesheet" href="{% static 'css/style.css' %}">
```
